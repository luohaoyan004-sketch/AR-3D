# AR Dance iOS Stable Final

This final version is optimized for iPhone/iPad Safari stability.

- Default page has no extra explanation text.
- It tries to request camera permission automatically.
- If iOS requires a user gesture, only one Start button is shown.
- It uses camera background + Three.js GLB animation overlay.
- It avoids WebXR on iOS to prevent black screen.

Deploy the entire folder to Netlify, Vercel, or GitHub Pages and open through HTTPS.
For troubleshooting, append ?debug=1 to the URL.
